Base de datos en SQL Server

Nota: La eleccion de SQL Server es completamente arbitraria, se puede usar cualquier otro gestor, sin embargo, si usted es nuevo respecto a gestores de base de datos, se recomienda utilizar este. Usted puede instalarlo desde aqui: Descargar SQL Server, seleccionando la opción gratuita Express.

Una vez instalado, debe crear un super usuario, asi como conectarse a un servidor (mayormente este ya viene predefinido, por lo que solo deberá darle Conectar).

Creando una nueva base de datos

Para crear una nueva base, basta con dar clic derecho sobre nuestro servidor. Posteriormente seleccionamos Nueva base de datos.

Luego de colocarle el nombre que deseemos, se nos creara nueestra base de datos. Sin embargo, una base de datos vacia de poco o nada sirve, por lo que le agregaremos tablas. Para ello, hacemos clic derecho sobre nuestra recien creada base de datos, y seleccionamos Nueva consulta:

Esto nos creara una seccion donde podremos escribir libremente. Aqui debemos crear las tablas que contendrá nuestra base de datos, asi como sus columnas y llaves primarias. Para una creación de tablas, se puede seguir el siguiente ejemplo:

CREATE TABLE [Nombre de tabla] ( [Columnas]... );

Para la columnas es importante especificar el nombre de la columna, su tipo, su tamaño (de ser necesario) e identificarla como llave primaria (en caso que lo sea); los tipos más usados en tablas son:

INTEGER Numero entero.
VARCHAR(#) Cadena de texto, siendo # el tamaño maximo de la misma.
FLOAT Numero flotante.
BOOLEAN Boleano
PRIMARY KEY No se trata de un tipo, pero es la palabra clave para identificar la llave primaria, obligatoria en toda tabla

El siguiente codigo, por ejemplo, muestra la creación de una tabla de nombre Estudiante, siendo id su llave primaria.

CRUD basico

Para ingresar un nuevo elemento a nuestra tabla, utilizamos el comando INSERT de la siguiente manera:

INSERT INTO [Nombre de Tabla] ([Nombre de las variables a ingresar, separado por coma]) VALUES ([Valores de las variables a ingresar, separada por coma]);

Por ejemplo, para ingresar un nuevo estudiante, hacemos:

INSERT INTO Estudiante(id, nombre, edad) VALUES (0, 'Jorge', 15);

Para obtener los elementos de una tabla, utilizamos el comando SELECT de la siguiente manera:

SELECT * FROM [Nombre de Tabla] WHERE([Setencia a cumplir]);

El comando WHERE es opcional, y nos permitira filtrar elementos especificos que cumplan con la sentencia dada, de otra forma, se retornaran todos los elementos de la tabla. Por ejemplo, para obtener todos los estudiantes de edad de 15 años, utilizamos:

SELECT * FROM Estudiante WHERE(edad = 15);

Para actualizar los datos de un elemento, utilziamos UPDATE de la siguiente manera:

UPDATE [Nombre de Tabla] SET [Variable que se desea cambiar] = [Nuevo valor] WHERE ([Sentencia]);

En este caso, el comando WHERE si es necesario, puesto que solo se modificarán aquellos elementos que cumplan con esta restricción. Por ejemplo, para actualizar la edad de un Estudiante cuyo id es igual a 5:

UPDATE Estudiante SET edad = 16 WHERE(id = 5);

Finalmente, para eliminar un elemento de la tabla hacemos uso de DELETE:

DELETE FROM [Nombre de Tabla] WHERE ([Sentencia]);

Una vez más, el comando WHERE es necesario para identificar el elemento, o elementos, que se desean eliminar de la base de datos de manera permanente. Por ejemplo, para eliminar a todos los estudiantes menores de 14 años:

DELETE FROM Estudiante WHERE(edad < 14);

Aunque existen muchas más sentencias y algoritmos referentes a SQL, para la creación de nuestra aplicación web, este CRUD basico bastará.
