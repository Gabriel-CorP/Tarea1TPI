Introdución a las cadenas de caracteres en VB.Net


Una cadena suele ser representada entre comillas dobles superiores ("palabra"), mientras que un carácter de esa cadena (un char) suele ser representado entre comillas simples ('p'). Por ejemplo:

char c = 'a';

char str[5] = "hola";

string palabra = 'hola Maria';

Generalmente para acceder a un carácter en una posición determinada se suele usar la forma variable posición como cuando se accede a un vector.

.length(): Nos devuelve el tamaño que tiene la cadena. El primer carácter de la cadena será el del índice 0. Junto con el método .length() podemos recuperar todos los caracteres de la cadena de texto. Si intentamos acceder a un índice de carácter que no existe el resultado que devolverá una excepción IndexOutOfBoundsException.

Manejo de caracteres:


Los métodos que nos permite manejar los caracteres de una cadena de texto, son una gran variedad. Se muestra algunos de ellos: Para ponerles en mayúsculas, minúsculas, quitarles los espacios en blanco, reemplazar caracteres.

String toLowerCase(): Convierte todos los caracteres de una cadena en minúsculas.
String toUpperCase(): Convierte todos los caracteres de una cadena en mayúsculas.
String trim(): Elimina todos los espacios en blanco de una cadena.
String replace(char oldChar, char newChar): Este método lo utilizaremos cuando lo que queramos hacer sea el remplazar un carácter por otro. Se reemplazarán todos los caracteres encontrados.

Tipos de cadenas de caracteres en VB.Net:

Las cadenas pueden ser segun la naturaleza: naturaleza dinámica o de naturaleza estática. En este segundo caso, el programador debe tener en cuenta que al recorrer la cadena los índices no se vayan de los límites previstos.

  • De naturaleza dinámica: pueden alterar su longitud durante el tiempo de ejecución.
  • De naturaleza estática: su longitud es fija a lo largo del tiempo de ejecución.

El final de la cadena se delimita de diferente manera en uno u otro caso:

  • Mediante un carácter de fin de cadena ('\0' en C) para las cadenas de tipo dinámico.
  • Mediante una propiedad de la cadena que delimite su longitud (length) para las de tipo estático.

Funciones de Cadenas de Caracteres:


Se describen diferentes formas de analizar y manipular cadenas. Algunos métodos son parte del lenguaje Visual Basic. Los métodos de Visual Basic .NET se utilizan como funciones inherentes al lenguaje.
Se muestra el uso habitual de un comando de manipulación de cadenas de Visual Basic .NET:

Dim aString As String = "SomeString"
Dim bString As String
bString = Mid(aString, 3, 3);

En el ejemplo anterior, la función Mid realiza una operación directa en aString y asigna el valor a bString.

Existen dos tipos de métodos en String: métodos compartidos y métodos de instancia. Que se muestran a continuación:

Método compartido: Es un método que se deriva de la propia clase String y no necesita una instancia de dicha clase para funcionar. Estos métodos se pueden calificar con el nombre de la clase (String) en vez de con una instancia de dicha clase.
El método String.Copy es un método estático, que actúa sobre una expresión dada y asigna el valor resultante a bString.

Dim aString As String
bString = String.Copy("A literal string")

Método de instancia: Se derivan de una instancia concreta de String y deben calificarse con el nombre de la instancia.
El método String.Copy es un método estático, que actúa sobre una expresión dada y asigna el valor resultante a bString. En este el método SubString es un método de la instancia de String es decir aString, realiza una operación en aString y asigna ese valor a bString.

Dim aString As String = "A String"
Dim bString As String
