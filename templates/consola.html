Programacion en consola


La programación en consola es de las más básicas y de las primeras a nivel de aprendizaje por la que se debería empezar para aprender en lenguajes de programación populares como Java, Python, C#, entre otros, el lenguaje de programación de Visual Basic .NET tampoco es la excepción, si bien su sintaxis es parecida a C#, podemos decir que en realidad es una mezcla entre la programación en C# con el IDE de Visual Studio y la programación de macros en Excel con Visual Basic.

Para comenzar a programar en consola con Visual Basic .NET basta con contar con un IDE para editar y compilar el código que escribamos, si bien se puede programar en cualquier editor de texto como inclusive desde el bloc de notas de Windows que no es una práctica recomendada, además se requiere un nivel más avanzado de conocimiento y dominio del lenguaje de programación, para efectos de aprendizaje se utilizara un IDE, en este caso un compilador en línea, que permitirá ver y comprender de mejor manera la explicación de cada punto que se desea explicar.

Introducir y mostrar datos en consola en bastante sencillo gracias a la clase Console. Para programar salidas en Visual Basic .NET basta con usar la instruccion Console.Write(Parametro), donde el parametro en la instruccion puede ser del tipo String,Char,Integer,entre otros.

Ejemplo de salida en Visual Basic .NET usando la funcion Write


Si queremos que de un salto de linea automaticamente, basta con usar la instruccion Console.WriteLine(Parametro), donde el parametro en la instruccion puede ser del tipo String,Char,Integer,entre otros.

Ejemplo de salida en Visual Basic .NET usando la funcion WriteLine


Para programar entradas en Visual Basic .NET basta con usar la instruccion valor = Console.ReadLine(), donde valor es una variable que debera coincidir con el parametro que se estara leyendo en la consola, veremos que en la consola el puntero se queda parpadeando (en caso de usar el modo interactivo de la consola de este sitio web), a la espera de que introduzcamos datos, si no pues debe ingresar el valor en la caja de texto llamada Stdin Inputs. Al pulsar Enter, seguirá a la siguiente línea. Si queremos concatenar algun valor con el texto mostrado, usamos el operador &.

Ejemplo de entradas en Visual Basic .NET


Los datos obtenidos mediante la funcion ReadLine pueden ser operados y tratos como cualquier otra variable que se ubiera declarado con un valor fijo, al igual que pueden ser almacenados posteriormente de nuevo en otra variable.

Ejemplo de operaciones con datos leidos con ReadLine en Visual Basic .NET


En la programacion por consola tambien se pueden utilizar estructuras de control (If) para validar condiciones o estructuras de ciclos repetitivos (For) y casi cualquier otra estructura que hallamos conocido a lo largo de este sitio web.

Ejemplo de estructuras de control y estructuras repetitivas con Entradas y Salidas (E/S) en Visual Basic .NET


Estas estructuras de control y estructuras repetitivas y en general cualquier estructura, nos sirven para controlar la logica de la ejecucion del programa, nos ayudan a delimitar el camino que tendra el usuario en el programa, asi como tambien poder interactuar con el de manera mas controlada.

En esta seccion se mostro de manera basica una introduccion a la programacion en consola en Visual Basic .NET, siendo en esta modalidad las operaciones mas importantes y utlizadas las operaciones de Entrada y Salida (E/S) y las estructuras de control mas ocupadas como lo son la estructura For y la estructura If, asi como tambien el manejo de variables para operaciones y calculos para su posterior utilizacion.
