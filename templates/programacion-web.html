Programación Web en VB .NET

El mundo ha progresado de manera asombrosa, y lo que antes era el standar de las aplicaciones, aplicaciones de escritorio, actualmente han evolucionado a aplicaciones web. Para afrontar esta nueva realidad, muchos lenguajes de programación se adaptaron y evolucionaron para permitir y facilitar el desarrollo de estas aplicaciones, siendo Visual Basic una de estas.

Y pese a parecer facil, el desarrollo de aplicaciones web tiene cierto grado de complejidad, por ello lo intentaremos de explicar de forma sencilla en los siguientes puntos:

Base de datos en SQL Server:
→ Creacion y CRUD basico de base de datos con SQL Server.
Vistas html:
→ Archivos html y css.
Conexion a la base de datos:
→ ASP.Net y SQLConection
API's:
→ Creacion y conexion a API's

Asi, con los conocimientos adquiridos en estos apartados, más el ya conocido por cursos anteriores, seremos capaz de crear nuestra propia aplicación web con conexion a base de datos.
