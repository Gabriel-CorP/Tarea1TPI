Arrays en VB.NET

Un array almacena una colección secuencial de tamaño fijo de elementos del mismo tipo. Los array se usan para almacenar una colección de datos, pero a menudo es más útil pensar en un array como una colección de variables del mismo tipo. Todos los array constan de ubicaciones de memoria contiguas. La dirección más baja corresponde al primer elemento y la dirección más alta al último elemento.

Creando arrays en VB.Net

Para declarar una matriz en VB.Net, usa la instrucción Dim. Por ejemplo:

Dim intData(30) 'Un array de 31 elementos
Dim strData(20) As String 'Array de 21 strings
Dim twoDarray(10, 20) As Integer 'Array bidimensional de integers
Dim ranges(10, 100) 'Array bidimensional

También puede inicializar los elementos del array mientras declara. Por ejemplo:

Dim intData() As Integer = {12, 16, 20, 24, 28, 32}
Dim names() As String = {"Karthik", "Sandhya", "Shivangi", "Ashwitha", "Somnath"}
Dim miscData() As Object = {"Hello World", 12d, 16ui, "A"}

Arrays dinámicos

Las Arrays dinámicos son Arrays que se pueden dimensionar y redimensionar según las necesidades del programa. Puede declarar un Array dinámico utilizando la sentencia ReDim
Sintaxis para la sentencia ReDim:

ReDim [Preserve] NombredelArray(NumeroDeElemetos)

La palabra clave Preserve ayuda a preservar los datos en una matriz existente cuando se cambia su tamaño.
NombredelArray: el nombre del array a redimensionar.
NumeroDeElemetos: especifica la nueva dimensión

Arrays multidimensionales

VB.Net permite arrays multidimensionales. Los arrays multidimensionales también se denominan matrices rectangulares.
Puede declarar una matriz bidimensional de Strings Así:

Dim ArrayBidimensional(10, 20) As String
