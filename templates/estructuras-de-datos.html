Estructuras de datos en VB .NET

Las estructuras de datos en programación son formas que ayudan a almacenar, organizar y utilizar eficientemente los datos. Visual Basic .NET cuenta con un conjunto predefinido de estructuras de datos o Colecciones, las cuales cuentan con propiedades y métodos comunes al manejo de dichas estructuras y los objetos que contienen.

Para utilizar estas colecciones, es necesario digitar "Imports System.Collections" al inicio del archivo de código fuente. Algunas de las colecciones incluidas en el espacio de nombres son:

ArrayList:
→ Colección ordenada de objetos que pueden indexarse individualmente.
Cola (Queue):
→ Colección de objetos donde el primer objeto que entra es el primer objeto que sale.
Pila (Stack):
→ Colección de objetos donde el primer objeto que entra es el último objeto que sale.
SortedList:
→ Colección de pares clave-valor donde se accede a los objetos mediante una clave o índice.

Cada página contiene un ejemplo de uso de dicha colección.
